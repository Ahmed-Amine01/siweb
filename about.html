About WTS

Ce projet est WhereTheSpeak (WTS). C'est un site qui permet de savoir dans quel pas on parle une langue. Si vous souhaitez accéder au code source du projet, il suffit de cliquer ici.


Pour ce projet, on utilise une requête Sparql pour récupérer les informations depuis wikidata. On utilise leaflet.js pour afficher la carte intéractive.
